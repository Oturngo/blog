股票即時追蹤清單(Google sheet 結合 Fugle API)【須有玉山富果證券戶】

發表於2021-07-05|更新於2021-07-05|tools
|字數總計:583|閱讀時長:1分鐘|閱讀量:

使用股票即時追蹤清單，不用打開看盤軟體也可以即時查看目前的持股損益，上班族可以安心使用XD

因這個追蹤清單需要申請 Fugle API token，要有富果證券戶才可以申請，若還沒有的可以點擊下方連結看線上開戶教學喔。
Fugle 富果線上開戶教學

會想用 Fugle API 的原因是因為若用證交所 API 取得股票資訊，上市、上櫃及興櫃得用不同 API 取得，而且也不確定這樣的取得方式是否合法(雖然網路上很多人都這樣做XD)，剛好自己有富果證券戶，感覺串它的API滿方便的。

Fugle API 可以取得上市、上櫃、興櫃的股票資訊喔

在使用清單前要先做以下兩件事情:
(1) 申請 Fugle API token
(2) 將清單建立副本

申請 Fugle API token

  1. 點擊下面的網址
    👉 https://developer.fugle.tw/

  2. 若已經開戶完成的，點擊畫面右上角的登入

  3. 先輸入手機或Email，接著再輸入密碼完成登入

  4. 登入完成會出現此畫面，等待約兩秒會自動跳轉回首頁

  5. 點擊畫面右上角的人頭，然後選取 API 權限/管理

  1. 因為我已經申請過了，所以畫面才會有一組 API token，你們只要點擊右下角的 **新增 API Token**，畫面就會跟我一樣了。

  2. 中間的 API token 等一下會用到，接下來要來建立副本。

將清單建立副本

  1. 點擊下方的清單連結
    清單連結

  2. 點擊 檔案 > 建立副本

  1. 自行修改副本名稱

  2. 建立完成後會自動開啟，這時候許多欄位是空白的，這是因為還沒填入 Fugle API token，所以是正常的喔!

  3. 點選下面的 "基本設定" 工作表，將剛剛申請的 API token 填入，另外手續費的折扣也填在這邊喔!

  4. 填好之後再切回 “管理清單” 工作表，將股票代碼 delete 掉再重新輸入，就會自動取得股票名稱跟股價囉!

恭喜你完成了屬於你自己的股票追蹤清單囉!
若在使用上有遇到任何問題的話，歡迎在下方留言~
